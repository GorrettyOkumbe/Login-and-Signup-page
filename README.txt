# 🔐 Authentication App (Login & Signup)

## 📌 Project Overview

This project is a simple web application that allows users to **sign up and log in** securely. It is built as a beginner-friendly project to demonstrate basic authentication concepts and frontend-backend interaction.

The application will provide users with forms to create an account and log in using their credentials.

---

## 🛠️ Technologies Used

The project will be developed using the following tools and technologies:

* Next.js – for building the frontend and backend (API routes)
* HTML – for structuring the web pages
* CSS – for styling the application
* Bootstrap – for responsive and modern UI components
* SQLite – for storing user data

---

## ✨ Features

* User Signup (Create Account)
* User Login (Authentication)
* Form validation (basic input checks)
* Responsive design using Bootstrap
* Simple backend API using Next.js

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate into the project folder:

```bash
cd auth-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
