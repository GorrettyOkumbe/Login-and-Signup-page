PROBLEM
Company XYZ had a webpage and could not keep track of the people accessing that same page.
Therefore I came up with this solution of designing a login and signup.

OBJECTIVE
To create a login and signup page that will help keep track of the people accessing the page.
Use a database to store these particular records.
For a person accessing the page for the first time they have to register all the necessary  details and create a password.
A person who has visited the page before only need to provide the username and password and the account is automatically retrieved.


Tools used
HTML
CSS
Bootstrap
Javascript
SQLite
